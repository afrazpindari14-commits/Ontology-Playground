# 🌐 Ontology-Playground - Visualize and build your data models

[![](https://img.shields.io/badge/Download-Ontology--Playground-blue.svg)](https://github.com/afrazpindari14-commits/Ontology-Playground/releases)

Ontology-Playground helps you learn how data models work. You can explore existing structures or build your own from scratch. The app runs directly in your web browser, which means your data stays on your computer. You do not need to install complex server software or manage databases to use this tool.

## 🛠️ What is this tool?

This application serves as a visual interface for knowledge representation. An ontology acts as a blueprint for data. It describes how different pieces of information relate to each other. Microsoft Fabric IQ uses these concepts to organize complex data environments. 

You can use this tool to:
* View a library of pre-built ontologies.
* Build new diagrams using a visual editor.
* Export your work into standard RDF/XML formats.
* Create links between concepts.
* Share your design diagrams with others.

The application remains fully static. This means it functions as a standalone file. It does not send your data to an external server. You keep full control over every file you create.

## 📥 Getting the software

You need to download the application package to your Windows computer to begin. Follow these installation steps to set up the software.

1. Go to the [official release page](https://github.com/afrazpindari14-commits/Ontology-Playground/releases).
2. Look for the section labeled Assets.
3. Click the link that ends with a .zip extension.
4. Save the file to your Downloads folder.

Once the download finishes, locate the file in your folder. Right-click the file and select Extract All. Choose a location on your hard drive for the files. Open the folder and double-click the file named index.html to launch the tool in your preferred web browser.

## 💻 System requirements

The application works on modern Windows operating systems. Ensure you meet these requirements:

* Windows 10 or Windows 11.
* A web browser like Microsoft Edge, Google Chrome, or Mozilla Firefox.
* At least 200 MB of free disk space.
* No internet connection required after the initial download.

## 📖 How to use the interface

The main window displays a canvas. This area holds your diagrams. Use the sidebar on the left to access your library of concepts.

### Starting a new project
Select the New Project button from the top menu. This action clears the canvas and prepares a blank space. You can name your project using the settings tab.

### Adding items
Drag boxes from the menu onto the canvas. Each box represents a concept. You can edit the text inside each box. Use the connector tool to draw lines between boxes. These lines define relationships between your concepts. 

### Exporting your work
The application supports the RDF/XML standard. Use the Export button to save your current diagram as a file. This file stores your data structure in a format that other software can read. This feature helps you move your work between tools or share your model with colleagues
